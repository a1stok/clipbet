# ClipBet

Instant prediction markets at live events. No app, no account, no friction.

## How It Works

**User Flow:** Scan > Bet > Leave

**Organizer Flow:** Scan > Create Event > Manage Dashboard > Leave

## Tech Stack (Demo Website)

- Apple App Clips for zero-install betting
- Built at Hack Canada

## Run Locally

```
npm install
npm run dev
```

Press Ctrl+'-' to zoom out if the page is too tall for your screen.
