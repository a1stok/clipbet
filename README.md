# ClipBet

Instant prediction markets at live events. No app, no account, no friction.

## How It Works

**Bettors:** Scan > Bet > Leave

**Organizers:** Scan > Create Event > Manage Dashboard > Leave

## Demo

[Watch the demo](https://youtube.com/shorts/yDGhNRkr158)

## Tech Stack

- Next.js, Tailwind CSS, Framer Motion
- Apple App Clips for zero-install betting
- Built at Hack Canada

## Run Locally

```
npm install
npm run dev
```
